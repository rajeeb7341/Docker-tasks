Schematic

  • Describes the content, not the layout
  • Meant to prent DIV-itis
  • Some of them: article, aside, details, summary, figure, figcaption, footer, header, hgroup, mark, meter, nav, progress, ruby, rt, rp, section, time, wbr
  • More Awesome Code Samples

Schematic Support

Data Attributes

  • Allows for any attribute in your HTML
  • Prefix your attribute with data-
  • Available via CSS/JavaScript
  • Example: <b data-ray="theawesome" data-clothing="he wears the best">
  • Even More Awesome Code Samples

Data Attribute Support

Geolocation

  • Not just for mobile
  • Requires user approval (and they can change their minds!)
  • Supports options for accuracy, caching, and timeout
  • Support for firing automatically on position change
  • Can fail
  • Super Cool Code Samples

Local(Session)Storage

  • Like cookies, but not
  • 5 megs (who could ever need more???)
  • Key/Value pairs, simple values (but JSON works great)
  • API via getItem, setItem, removeItem
  • File based
  • Super Cool Code Samples

LocalStorage Support

IndexedDB Storage

  • NoSQL/Object based persistance
  • Much deeper (and therefore more complex) than localStorage.
  • Stop using WebSQL - this is the future (whether you like it or not).
  • Super Cool Code Samples
